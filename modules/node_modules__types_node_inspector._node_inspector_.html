open

  • open(port?: number, host?: string, wait?: boolean): void
    • Defined in node_modules/@types/node/inspector.d.ts:2709

    Activate inspector on host and port. Equivalent to node --inspect=[[host:]port], but can be done programmatically after node has started.

    If wait is true, will block until a client has connected to the inspect port and flow control has been passed to the debugger client.

    See the security warning regarding the hostparameter usage.

    Parameters

    • Optional port: number
    • Optional host: string
    • Optional wait: boolean
